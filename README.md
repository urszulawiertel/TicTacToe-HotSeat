# TicTacToe Hot-Seat (SwiftUI)

A simple Tic-Tac-Toe (3x3) iOS app built with SwiftUI.

## Features
- 3x3 board with X / O turns
- Win & draw detection
- Highlight winning line
- Score tracking (X / O)
- Reset board / reset score
- Hot-seat timer (10s per move)
- Game engine extracted from UI (`TicTacToeEngine`)
- Unit tests for core game logic (`XCTest`)

## Tech stack
- Swift
- SwiftUI
- ObservableObject (`@StateObject`, `@Published`)
- XCTest

## How to run
1. Open the project in Xcode
2. Select an iOS Simulator
3. Run (`Cmd + R`)

## Tests
Run unit tests:
- `Cmd + U`
